# Week 4 - Constructors and Class Design

## Overview

This week focuses on constructors, object initialization, and designing classes with meaningful functionality.

## Learning Objectives

* Understand constructor overloading
* Initialize objects efficiently
* Perform calculations using classes
* Design reusable class structures

## Experiments

### Experiment 1: Employee Class with Constructors

Implement multiple constructors and getter/setter functions.

### Experiment 2: Circle Class

Calculate area, diameter, and circumference.

### Experiment 3: Inventory Class

Store inventory details and calculate inventory values.

## Concepts Covered

* Constructor Overloading
* Object Initialization
* Encapsulation
* Validation
* Mathematical Computations

## Language Used

C++

## Outcome

Students learned how constructors improve object creation and simplify program design.
